# МИНС. 2026 год. ЛР 1-4

Репозиторий предназначен для отчетов по ЛР курса МИНС в 2026 году.

Логика такова.
1. Сделать `fork` этого репозитория
2. В папке для Вашей группы создать папку со своим FIO в формате `фамилияИО`
пример, Иванов Сергей Кириллович создает папку `ivanovsk`
4. В собственной папке создать папки `lr1`, `lr2` и тд.
5. Залить в папки Лабораторные работы 1-4
6. Сделать Пулл реквест

Примечание, если нет разделенных ЛР по коду (Первая и Вторая в одном проекте), создать папку в названиии которой будет указано количество этих лабораторных работ (`lr1-2`)
