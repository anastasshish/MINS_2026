## Что ты такое?

**SOS_AL** (*SOS Algorithmic Language*, в народе — просто **«Сос»** или **«СосАЛ»**) — учебный алгоритмический язык с ключевыми словами на русском языке с поддержкой ООП, модулей, разделения на несколько файлов и исключений. 

Под капотом — интерпретатор на **Python**, который:

- лексически разбирает исходники (`.сос`);
- строит AST;
- выполняет код с классами, наследованием, исключениями и модулями;
- не крашит среду разработки из-за файла больше 500 строк.

## ИС «Аптека» — главный пациент проекта

В папке `examples/apteka/` живёт **информационная система аптеки** — консольное приложение, которое умеет:

- хранить препараты (рецептурные и безрецептурные);
- продавать их с проверкой рецепта, срока годности и остатков на складе;
- искать, сортировать и следить за «истекающими» лекарствами;
- вести журнал продаж в текстовых файлах.

Благодаря тому, что этот язык **не является** КуМиром, в нём есть возможность реализовать все доступные паттерны проектирования. Например, такие как уже реализованные:

| Паттерн | Зачем в аптеке |
|---|---|
| **Repository** | Препараты и продажи лежат аккуратно, как на полке |
| **Factory** | Создаём лекарства по типу — рецептурное или нет |
| **Strategy** | Поиск и сортировка меняются без переписывания половины кода |
| **Observer** | События пишутся в консоль — «продано», «ошибка», «проиграл» |
| **Chain of Responsibility** | Валидация: формат → срок → склад → рецепт |
| **Unit of Work** | Транзакции, чтобы продажа не зависла на полпути |

## Структура проекта

```
SOS_AL/
├── sos/                  # Ядро интерпретатора (лексер, парсер, рантайм)
├── main.py               # Точка входа
├── tests/                # Тесты pytest
├── examples/apteka/      # ИС аптеки на языке Сос
├── vscode-sos/           # Расширение VS Code: подсветка синтаксиса .сос
└── .vscode/              # Настройки редактора для проекта
```

---

## Быстрый старт

**1. Запустить аптеку:**

```bash
python main.py examples/apteka/apteka.сос
```

**2. Погонять тесты:**

```bash
python -m pytest tests/ -v
```

**3. Подсветка синтаксиса в VS Code:**

Откройте папку `vscode-sos/` и установите расширение локально (*Install from VSIX* или через *Developer: Install Extension from Location*).
